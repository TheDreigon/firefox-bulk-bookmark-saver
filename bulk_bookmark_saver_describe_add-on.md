# Bulk Bookmark Saver

## Summary

Bulkmark: save multiple tabs as bookmarks at once and close windows while preserving all tabs in organized folders - perfect for managing dozens of open tabs efficiently. Need to close and bookmark dozens of tabs in an instant? Use bulkmark.

## Description

Bulkmark makes managing dozens of open tabs effortless with powerful bulk bookmarking features:

**Key Features:**
- Click the extension icon to save all tabs in the current window and close it
- Right-click selected tabs to bookmark them in bulk
- Choose where to save bookmarks or set a default folder
- Create new bookmark folders directly from the popup
- All tabs are automatically organized in dated folders for easy retrieval

## How to Use

1. Click the extension icon to save all tabs and close the window
2. Or select multiple tabs (Ctrl+click), right-click and choose "Save selected tabs to bookmarks"
3. Choose a destination folder or create a new one
4. Your tabs are saved in an organized folder with the current date and time

Perfect for research, studying, or anyone who tends to accumulate too many tabs! Also perfect if you need to close and bookmark dozens of tabs in an instant!

## Privacy

This extension does not collect, store, or transmit any user data. All operations are performed locally in your browser.

## Note

This extension is designed for Firefox Desktop only and is not yet compatible with Firefox Android.
